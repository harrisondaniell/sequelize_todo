## Instalação

Você pode clonar o repositório e rodar:

```bash
npm install
```

Depois clonar o arquivo `.env.example` para `.env.local`:

```bash
cp .env.example .env.local
```

E alterar as variáveis de ambiente.

## Uso

Para rodar o projeto, utilize o comando padrão:

```bash
npm run dev
```
